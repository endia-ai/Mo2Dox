## Functions

#### **conv1d**

```python
conv1d(arg0: Array, kernel: Array, bias: Array, stride: Int, padding: Int, dilation: Int, groups: Int) -> Array
```

<details>
<summary>Applies a 1D convolution over an input signal composed of several input planes.</summary>

###### Args

- ``arg0``: ``Array``
  Input tensor of shape (batch_size, in_channels, length).

- ``kernel``: ``Array``
  Convolution kernel of shape (out_channels, in_channels // groups, kernel_size).

- ``bias``: ``Array``
  Bias tensor of shape (out_channels).

- ``stride``: ``Int``
  Stride of the convolution.

- ``padding``: ``Int``
  Zero-padding added to both sides of the input.

- ``dilation``: ``Int``
  Spacing between kernel elements.

- ``groups``: ``Int``
  Number of blocked connections from input channels to output channels.

###### Returns
- ``Array`` - Output tensor of shape (batch_size, out_channels, output_length).


</details>
