## Functions

#### **sum**

```python
sum(arg0: Array) -> Array
```

<details>
<summary>Computes the sum of the input array along all axes.</summary>

###### Args

- ``arg0``: ``Array``
  The input array.

###### Returns
- ``Array`` - An array containing the sum of the input array along all axes.

**Examples:**
```python
 a = Array([[1, 2], [3, 4]])
 result = sum(a)
 print(result)
```

**Note:**
This function supports:
- Automatic differentiation (forward and reverse modes).
- Complex valued arguments.
