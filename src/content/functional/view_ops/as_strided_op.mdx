###### Args

  The current array to store the result (modified in-place).

- ``args``: ``List[Array]``
  The array on which the as_strided view is created.

**Note:**


<details>
<summary>Computes the vector-Jacobian product for the as_strided operation.</summary>

###### Args

- ``primals``: ``List[Array]``
  A list containing the primal input array.

- ``grad``: ``Array``
  The gradient of the output with respect to some scalar function.

- ``out``: ``Array``
  The output of the forward pass.

###### Returns
- ``List[Array]`` - A list containing the gradient with respect to the input.

**Note:**
The vector-Jacobian product for as_strided is computed by calling the inverse operation as_strided_inv.


</details>
