- ``x``: ``Array``
  The input array.

- ``dims``: ``List[Int]`` (default: ``List(-2, -1)``)

- ``norm``: ``String`` (default: ``String("backward")``)
  The normalization mode.

###### Returns
- ``Array`` - The 2-dimensional FFT of the input array.
