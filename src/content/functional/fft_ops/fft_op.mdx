#### **fft**

```python
fft(x: Array, dim: Int = -1, norm: String = String("backward")) -> Array
```

<details>
<summary>Compute the n-dimensional FFT.</summary>

###### Args

- ``x``: ``Array``
  The input array.

- ``dim``: ``Int`` (default: ``-1``)
  The dimension along which to compute the FFT.

- ``norm``: ``String`` (default: ``String("backward")``)
  The normalization mode.

###### Returns
- ``Array`` - The n-dimensional FFT of the input array.
